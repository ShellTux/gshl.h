# GSHL

GSHL is a single header library to easily port this library to any project.

Just copy and paste a single file, in stb-style.

## Examples

### Printing

Print to the console or to a file like printf
you can write custom format specifiers

./examples/print.c:
```c
!include ./examples/print.c
```

### Hash Table

./examples/hash-table.c:
```c
!include ./examples/hash-table.c
```

### Logging

./examples/log.c:
```c
!include ./examples/log.c
```

### Tagged Union

If you want rust enums in c look at `./examples/tagged-union.c`:
```c
!include ./examples/tagged-union.c
```

### Option

If you want rust option in c look at `./examples/option.c`:
```c
!include ./examples/option.c
```

### Tcp Server/Client

#### Server

```c
!include ./examples/tcp/server.c
```

#### Client

```c
!include ./examples/tcp/client.c
```
